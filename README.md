# TechNova Solutions - Responsive Business Website

## 📌 Project Overview

TechNova Solutions is a modern, responsive business website developed using HTML5, CSS3, and JavaScript. The website showcases company information, services, and a contact page with form validation. The project demonstrates responsive web design principles and front-end web development best practices.

---

## 🎯 Project Objectives

- Develop a fully responsive multi-page website.
- Create a clean and modern user interface.
- Implement navigation between multiple pages.
- Add JavaScript form validation.
- Optimize the website for desktop and mobile devices.
- Deploy the project using GitHub Pages.

---

## ✨ Features

- Responsive Navigation Bar
- Hero Banner
- About Company Section
- Services Section
- Contact Form with Validation
- Responsive Design
- Social Media Footer
- Font Awesome Icons
- Modern UI Design

## 🚀 Setup Instructions

1. Download or clone the repository.
2. Open the project in Visual Studio Code.
3. Install the Live Server extension.
4. Open `index.html` using Live Server.
5. Browse all pages.
